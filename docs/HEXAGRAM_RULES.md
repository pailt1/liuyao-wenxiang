# 六爻排盘规则

## 六爻生成顺序

六爻必须从下往上生成。

1. 第一次摇卦：初爻，最下面
2. 第二次摇卦：二爻
3. 第三次摇卦：三爻
4. 第四次摇卦：四爻
5. 第五次摇卦：五爻
6. 第六次摇卦：上爻，最上面

## 硬币规则

正面 = 3  
反面 = 2

三枚硬币相加：

| 点数 | 名称 | 爻象 | 是否动爻 | 变化 |
|---|---|---|---|---|
| 6 | 老阴 | 阴爻 | 是 | 阴变阳 |
| 7 | 少阳 | 阳爻 | 否 | 不变 |
| 8 | 少阴 | 阴爻 | 否 | 不变 |
| 9 | 老阳 | 阳爻 | 是 | 阳变阴 |
