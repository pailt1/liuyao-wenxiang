# 六爻问象

六爻问象是一款结合传统六爻文化与 AI 解读的微信小程序。

用户可以输入问题，通过电子硬币摇卦，系统自动生成六爻，计算本卦、变卦、动爻，并调用 AI 进行辅助解读。

## 核心功能

- 输入问题
- 电子硬币摇卦
- 自动生成六爻
- 自动识别本卦
- 自动识别变卦
- 自动识别动爻
- AI 辅助解卦
- 历史记录保存
- 结果分享

## 项目原则

1. 程序负责排盘，AI 负责解释。
2. 六爻必须从下往上排列。
3. 不输出绝对化预测。
4. 不制造焦虑。
5. 所有解读仅供传统文化学习与自我反思参考。

## 技术栈

- 微信原生小程序
- JavaScript
- 微信云开发
- Node.js 云函数
- 云数据库
- 大模型 API
