# 产品功能文档

## 产品名称

六爻问象 / 象问

## 产品定位

一款结合传统六爻文化与 AI 解读的微信小程序。

## 目标用户

- 对易经、六爻、传统文化感兴趣的人
- 想通过卦象辅助思考问题的人
- 想记录问题和解读结果的人

## MVP 功能

### 1. 首页

用户输入问题，选择问题类型。

问题类型包括：

- 感情
- 学业
- 考试
- 事业
- 求职
- 选择
- 其他

### 2. 摇卦页

模拟三枚硬币摇六次。

每次生成一爻。

六次完成后得到完整六爻。

### 3. 结果页

展示：

- 用户问题
- 起卦时间
- 六爻图形
- 本卦
- 变卦
- 动爻
- AI 解读
- 现实建议
- 免责声明

### 4. 历史记录页

展示用户过去的起卦记录。

### 5. 详情页

查看某一次起卦的完整结果。
